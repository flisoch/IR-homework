# Запуск
В терминале

1. ```$julia```

2. ```julia> ]```

3. ```(v1.0) pkg> add https://github.com/flisoch/IR-homework```

4. Обратно в Julia REPL, нажав backspace

5. ```using Pluto```

6. ```Pluto.run()```

7. В плуто открыть файл ```/<path-to-project>/src/demo.jl```
